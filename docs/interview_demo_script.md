# K-APT AX Control Tower 인터뷰 시연 스크립트

## 30초 프로젝트 소개

K-APT는 관리비를 공개하지만 개별 단지가 어떤 비용을 왜 먼저 검토해야 하는지까지 제시하지 않습니다. 이 프로젝트는 공개데이터를 비교단지, 이상징후, 검토 가설, 실행 과제와 증빙 요청으로 전환한 AX Advisory 의사결정 지원 시스템입니다.

## 5분 시연 순서

### 1. Executive Overview — 40초

- 대상 단지의 전체 비용과 지표상 기회금액 설명
- 기회금액은 확정 절감액이 아니라 검토 우선순위임을 강조
- 비용항목별 격차와 이상징후 추이 확인

### 2. Peer Benchmark — 50초

- 단순 지역 평균 대신 구조적으로 유사한 30개 단지 사용
- 세대수, 사용승인연도, 구조 유사도와 관측률 설명
- 비교군을 설명 가능하게 설계했다는 점 강조

### 3. Cost Driver & Trend — 50초

- 대상 단지와 비교군의 월별 세대당 관리비 비교
- 비용항목별 기대 하단·중앙·상단 확인
- 일회성 변동과 구조적 격차를 구분

### 4. Anomaly Explorer — 50초

- 이상징후의 크기, 지속성 및 탐지 근거 설명
- 이상징후는 부정 판정이 아니라 추가 검토 신호임을 강조

### 5. Advisory Action Center — 70초

- 분석 신호를 담당자, 기간, KPI가 있는 실행 과제로 전환
- 계약서, 원장, 운영이력 등 증빙 요청 연결
- 모든 조치는 사람 승인 후 실행되도록 통제

### 6. Model QA와 마무리 — 40초

- 데이터 행 수, 관측률, 모델 입력 및 결과 정합성 검증
- 공개데이터의 한계와 Responsible AI 원칙 설명
- AX의 가치는 예측만이 아니라 검토와 실행 프로세스 연결에 있다고 정리s
