# K-APT AX Control Tower

AI-powered apartment management cost, procurement, and long-term repair decision platform.

## Problem

K-apt provides public management information, but apartment management offices,
resident councils, and residents still face difficulty converting fragmented data
into evidence-based decisions and concrete actions.

## Solution

K-APT AX Control Tower combines public apartment data, analytical models,
Power BI, and an AI Action Center to:

1. Identify unexplained management cost gaps
2. Detect procurement review signals
3. Forecast long-term repair reserve shortfalls
4. Generate evidence-based management actions

## Primary Users

- Apartment resident council
- Apartment management office

## Supporting Users

- Residents
- Property management companies
- Local government supervisors

## Architecture

- Python: data ingestion, transformation, analytics, and AI
- Power BI: management decision dashboard
- Streamlit: AI Action Center
- GitHub: version control and portfolio documentation
